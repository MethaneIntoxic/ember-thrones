# Game Design - Ember Thrones

## Pillars
- Dragon-themed high-drama slot experience.
- Ember Lock hold-and-respin as core feature.
- Four-tier jackpot ladder always visible.
- Free Quest retrigger loop as secondary feature.
- Three deterministic mini-games for retention bursts.

## Core Loop
1. Place bet and spin.
2. Resolve paylines and trigger checks.
3. Enter Ember Lock if 6+ orbs appear.
4. Enter Free Quest on scatter threshold.
5. Optionally enter mini-games from trigger windows.
6. Update wallet, progression, and telemetry.

## Feature Cadence Targets
- Ember Lock: every 50-60 spins.
- Free Quest: every 80-140 spins.
- Bonus retriggers: 15-25% during feature sessions.
- Small perceived wins: every 8-12 spins.

## Mini-Games
- Lantern Pick: value reveal choice game.
- Sky Path: risk-reward path game.
- Wyrm Duel: timing-based pressure game.
