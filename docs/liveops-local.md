# Local LiveOps Plan

## Seasonal Cadence
- Season length: 12 weeks.
- Additive updates through local config packs.
- Feature toggles controlled by server config endpoint.

## Event Strategy
- Weekly mini events adjust cosmetic rewards and challenge goals.
- Monthly tuning patch refreshes cadence and multiplier weighting.
- Preserve core acceptance gates during all events.

## Telemetry Review
- Daily review: spin count, trigger rates, payout drift.
- Weekly review: session length and mini-game usage.
- Release patch only if simulation and acceptance gates remain green.
